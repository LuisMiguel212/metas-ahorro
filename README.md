# Metas de Ahorro

Aplicación para gestionar metas de ahorro.

## Funciones

- Crear metas
- Registrar dinero disponible
- Registrar fechas de pago
- Calendario interactivo
- Gráficas de progreso
- Tema oscuro
- Exportar e importar datos

## Tecnologías

- HTML
- CSS
- JavaScript
- FullCalendar
- Chart.js
